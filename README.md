<p align="center">
    <b>Code Snippet</b> is an Angular component to display syntax highlighting to code snippets.
</p>

## Installation

Install using a package manager:

```shell
npm i ngx-code-snippet
```

or

```shell
yarn add ngx-code-snippet
```
