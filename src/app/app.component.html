Code Snippet

Syntax highlighting for Angular

Code Snippet is an Angular component to display code snippets with syntax highlighting.

Installation

Install using a package manager: or

Usage

First, import the module. Next, set up a config property. You can read all annotation types on Rough Notation's official documentation. Lastly, add component in the HTML.

Directives

Optionally, override the reset flag to avoid re-animations. Delay, set to 1000 milliseconds (1 second) by default, can also be overridden.
